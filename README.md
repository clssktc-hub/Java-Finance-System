Java 個人理財分析系統 (Personal Finance System)

這是一個使用 Java 開發的文字介面（Console-based）理財工具。旨在幫助使用者進行基本的財務健康診斷、投資試算以及貸款評估。

功能特色
本系統包含五大核心功能：

1.  **財務健康檢查**：輸入過去 6 個月的收支，自動計算儲蓄率並分析最高/最低儲蓄月份。
2.  **單筆投資計算**：基於複利公式，計算單筆資金在特定年限後的未來價值。
3.  **定期定額模擬**：模擬每月固定投入資金的複利效果，並顯示年度帳戶價值。
4.  **投資組合評估**：輸入三支股票的權重與預期報酬，分析整體風險水平（高/中/低）。
5.  **房貸計算**：使用本息平均攤還法，計算每月房貸支出並列出還款明細。

使用技術

* **Language**: Java SE
* **Concepts**: 
    * 基本語法與迴圈控制 (Loops, Switch-Case)
    * 陣列操作 (Arrays)
    * 數學運算 (Math Class)
    * 格式化輸出 (String Formatting)

1. 確保電腦已安裝 Java (JDK)。
2. 下載 `PersonalFinanceSystem.java`。
3. 開啟終端機 (Terminal) 或 CMD，編譯程式：
   ```bash
